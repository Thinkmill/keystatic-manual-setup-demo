# Keystatic Blank Template

This is a blank template using Keystatic as a CMS.

It uses:

- Keystatic
- NextJS

## Getting started

Here are a set of instructions to get Keystatic running locally

### Prerequisites

You will need to have pnpm installed
```
npm install -g pnpm
```

### Installing
```
pnpm i
```

Run the development server
```
npm run dev
```
